# Proiect Licență

Acest depozit conține sursele pentru un proiect de licență axat pe implementarea hardware (FPGA) și software a unor algoritmi de Machine Learning.

## Structura Proiectului

* **DecisionTreeImplementation/** - Implementarea algoritmului de tip arbore de decizie (Decision Tree).
* **NeuralNetwork/** - Implementarea unei rețele neuronale, în mai multe variante.
* **JailbreakDetection/** - Implementarea unui model de Random Forest pentru Jailbreak Detection, în mai multe variante.
* **SQLInjection/** - Implementarea unui model de Random Forest pentru detecția atacurilor de tip SQL Injection.
* **TestConnect2FPGA/FPGA/** - Folosit pentru a testa placa FPGA pentru prima dată
* **Rezultate/** - Metricele obținute în urma testelor.
